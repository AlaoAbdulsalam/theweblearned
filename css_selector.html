CSS Selector

In CSS there are many selector which selects HTML element you want to style, we will be explaining them below:

The CSS Element Selector

This selector selects HTML element based on the element name:

Example:

In the example below the h1 element is selected in the CSS:

    
       <!DOCTYPE html>
       <html>
       <head>
       <title>CSS Tutorials</title>
        <style>
            h1 {
                text-align: right;
                color: yellow;
            }

        </style>
       </head>
       <body>         
       
         <h1>This is my first heading</h1>
        
        </body>
       </html>

The CSS Class Selector

This selector selects specific HTML element. It is usually added in the opening tag.

To select a particular class selector in the CSS, write a period(.) character followed by name of the class. class name cannot start with a number

Example:

In the example below, the HTML element with the class name (paragraph) will be selected and centered with a red color:

                 
                    <!DOCTYPE html>
                    <html>
                    <head>
                    <title>CSS Tutorials</title>
                     <style>
                         .paragraph {
                             text-align: center;
                             color: red;
                         }
             
                     </style>
                    </head>
                    <body>         
                    
                      <p class="paragraph">A paragraph aligned to the center</p>
                     
                     </body>
                    </html>

HTML elements can also have more than one class name.

Example:

In the example below, the <p> element will be styled according to the "paragraph" and the "size" class

                 
                    <!DOCTYPE html>
                    <html>
                    <head>
                    <title>CSS Tutorials</title>
                     <style>
                         .paragraph {
                             text-align: center;
                             color: red;
                         }

                         .size {
                            font-size: 30px;
                            font-family: roboto;
                            font-weight: 300;
                         }
             
                     </style>
                    </head>
                    <body>         
                    
                      <p class="paragraph size">A paragraph with two classes</p>
                     
                     </body>
                    </html>

The CSS Id Selector

This selector selects specific HTML element. It is usually added in the opening tag.

There is no much different between the Id selector and the class selector except that the Id selector has the hash(#) tag before its name and the class selector has the period(.) before its name.

To select a particular class selector in the CSS, write a hash tag(#) character followed by name of the id. Id name cannot start with a number

Example:

In the example below, the HTML element with the id name (paragraph) will be selected and aligned to the right with a red color:

                  
                     <!DOCTYPE html>
                     <html>
                     <head>
                     <title>CSS Tutorials</title>
                      <style>
                          #paragraph {
                              text-align: right;
                              color: red;
                          }
              
                      </style>
                     </head>
                     <body>         
                     
                       <p id="paragraph">A paragraph aligned to the right</p>
                      
                      </body>
                     </html>

The CSS universal Selector

The universal selector selects all HTML elements in a page.

Example:

In the example below, the HTML element with the id name (paragraph) will be selected and aligned to the right with a red color:

                  
                     <!DOCTYPE html>
                     <html>
                     <head>
                     <title>CSS Tutorials</title>
                      <style>
                          * {
                              font-size: 20px;
                          }
              
                      </style>
                     </head>
                     <body>         
                     
                       <p>A paragraph with a background-color of blue</p>
                      
                      </body>
                     </html>

The CSS grouping Selector

The grouping selector selects all the HTML elements with the same style definitions.

If you want to style much HTML element in the same way, we suggest using CSS grouping selector. To select multiple element just add comma after one element

Example:

In the example below, we grouped all the selectors:

                  
                     <!DOCTYPE html>
                     <html>
                     <head>
                     <title>CSS Tutorials</title>
                      <style>
                          p, h1, h3{
                              font-family: roboto;
                              text-align: center;
                          }
              
                      </style>
                     </head>
                     <body>         
                     
                       <p>A paragraph with a background-color of blue</p>
                       <h1>A paragraph with a background-color of blue</h1>
                       <h3>A paragraph with a background-color of blue</h3>
                      </body>
                     </html>
