What is Multimedia?

A multimedia have different format, it can be music, sound, images, videos, records, films, audios, animation and so on.

Webpages usually contain one or more multimedia elements which usually have different types and formats.

Multimedia Formats

Multimedia elements such as (video or audio) are stored in a media files.

You can know the type of a file by looking at the file extension.

They usually have formats and extension like: .wav, .mp3, .mp4, .avi, .wmv and .mpg.

Video Formats

The MP4, WebM, and Ogg formats are all supported by HTML.

The MP4 format is used and recomended by YouTube.

Format File Description

MPEG

.mpg, .mpeg

MPEG: Means (Moving Pictures Expert Group). It was the first popular video format on the web. It is not supported anymore in HTML.

AVI

.avi

AVI: Means (Audio Video Interleave). It was developed by Microsoft. It is commonly used in video cameras and TV hardware. It also plays well on windows computers, but not in the web browsers.

WMV

.wmv

WMV: Means (Windows Media Video). It was developed by Microsoft. It is commonly used in video cameras and TV hardware. It also plays well on windows computers, but not in the web browsers.

QuickTime

.mov

QuickTime. It was developed by Apple. It is commonly used in video cameras and TV hardware. It also plays well on Apple computers, but not in the web browsers.

RealVideo

.rm, .ram

RealVideo. It was developed by Real Media to allow video streaming with low bandwidths. It does not play well in the web browsers.

Flash

.swf, .flv

Flash. It was developed by Macromedia. It often requires an extra component(plug-in) to play in the web browsers.

Ogg

.ogg

Theora Ogg. It was developed by Xiph.Org Foundation. It is supported by HTML.

WebM

.webm

WwbM. It was developed by Mozilla, Opera, Adobe and Google. It is supported by HTML.

MPEG-4 OR MP4

.mp4

MP4 was developed by the Moving Pictures Expert Group. It is commonly used in video cameras and TV hardware. It is supported by all browsers and recommended by YouTube.

Common Audio Formats

MP3 is the best format for compressed recorded music. The term MP3 has become synonymous with digital music.

Format File Description
MIDI .mid, .midi MIDI stands for Musical Instrument Digital Interface. MIDI files do not contain sound, but digital notes that can be played by electronics. It plays well on computers and music hardware, but not in web browsers.
RealAudio .rm, .ram RealAudio. It was developed by Real Media which allows the streaming of audio with low bandwidths and does not play well in web browsers.
WMA .wma WMA stands for Window Media Audio. It was developed by Microsoft. It plays well on windows computer but not in the web browser.
AAC .aac AAC stands for Advanced Audio Coding. It was developed by Apple which serves as default format for iTunes. It plays well on Apple computers but not in the web browser.
WAV .wav WAV was developed by IBM and Microsoft. It plays well on Makintosh, windows and Linux operating systems. WAV is fully supported by HTML
Ogg .ogg Ogg was developed by Xiph.Org Foundation. Ogg is fully supported by HTML.
MP3 .mp3 MP3 was developed by the Moving Pictures Expert Group. MP3 is the most popular format for music players. Combines good compression (small files) with high quality. Supported by all browsers.
MP4 .mp4 MP4 was developed by the Moving Pictures Expert Group. MP4 is a video format, but can also be used for audio. Supported by all browsers.
