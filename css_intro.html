What is CSS

  • CSS stands for "Cascading Style Sheet"
  • CSS is used for styling HTML document such as colors, font size and much more
  • CSS makes a webpage look attractive and user-friendly
  • CSS tells the browser how HTML element should be displayed
  • CSS is connected to HTML using the <link> element.

Why use CSS

CSS is a Cascading Style Sheet used for styling a webpage, with CSS you can style a webpage, you can style multiple pages with just a CSS file.

Example:

    
       <!DOCTYPE html>
       <html>
       <head>
       <title>CSS Tutorials</title>
        <style>
            body {
                background-color: black;
            }

            p {
                font-size: 18px;
                text-align: center;
                color: white;
            }

            h1 {
                font-size: 35px;
                text-align: center;
                color: white;
            }

        </style>
       </head>
       <body>         
       
         <h1>This is my first heading</h1>
         <p>This is my first paragraph</p>
       
        </body>
       </html>

HTML was created not to describe how it content is displayed, it only describe the content of the webpage

When tags like , and color attributes were added to the HTML 3.2 specification, it started a nightmare for web developers. Development of large websites, where fonts and color information were added to every single page, became a long and expensive process.

To solve this problem, the World Wide Web Consortium (W3C) created CSS. CSS removed the style formatting from the HTML page!
