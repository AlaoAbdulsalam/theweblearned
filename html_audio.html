HTML Audio

The HTML <audio> element is used to play an audio on a web page.

Example

    
       <!DOCTYPE html>
       <html>
       <head>
       <title>HTML Tutorials</title>
       </head>
       <body>         
       <audio controls>
         <source src="horse.ogg" type="audio/ogg">
         <source src="horse.ogg" type="audio/ogg">
         Your browser does not support the audio tag.
       </audio>
        </body>
       </html>

Explanations

The controls attribute add audio controls such as play, pause, and volume.

The second <source> element allows you to specify alternative audio files which the browser may choose from. The browser will use the first recognized format.

The text between the <audio> and <audio> tags will only be displayed if the browser do not support the format of the <audio> element.

HTML <audio> Autoplay

If you want to start an audio automatically, add the autoplay attribute to the <video> element:

Example

    
       <!DOCTYPE html>
       <html>
       <head>
       <title>HTML Tutorials</title>
       </head>
       <body>         
       <audio controls autoplay>
         <source src="horse.ogg" type="audio/ogg">
         <source src="horse.ogg" type="audio/ogg">
         Your browser does not support the audio tag.
       </audio>
        </body>
       </html>

Note: In most cases, chromium browsers do not allow autoplay. However, you can use muted attribute after the autoplay attribute in the <video> element.

Example

    
       <!DOCTYPE html>
       <html>
       <head>
       <title>HTML Tutorials</title>
       </head>
       <body>         
       <audio controls autoplay muted>
         <source src="horse.ogg" type="audio/ogg">
         <source src="horse.ogg" type="audio/ogg">
         Your browser does not support the audio tag.
       </audio>
        </body>
       </html>
