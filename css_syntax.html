CSS Syntax

CSS consists of a selector and a declaration block. Description below:

Explanations

  • The selector block point to the name of the element you want to style. For example you want to style the h1 element which is the selector.

  • The declaration block is divided into 2 sections: the property and the value.

  • In the example above the color is the property while the blue is the value.

  • The color is the type of effect you want to add to the h1 element while the blue is the effect you added to the h1 element.

  • Every declaration ends with a semi-colon. You can add more than one declaration to a CSS.

  • The declaration are surrounded in a curly brace.

Example:

In the example below the h1 element will be aligned to the right with a color of yellow:

    
       <!DOCTYPE html>
       <html>
       <head>
       <title>CSS Tutorials</title>
        <style>
            h1 {
                text-align: right;
                color: yellow;
            }

        </style>
       </head>
       <body>         
       
         <h1>This is my first heading</h1>
        
        </body>
       </html>

Explanation:

  • The h1 in the CSS is the selector pointing to the <h1> element in the HTML

  • The text-align is the property to the h1 element, while the right is the value to the h1 element

  • color is the property while the red is the value
