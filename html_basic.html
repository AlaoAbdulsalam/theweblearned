HTML Basic

In this chapter we will show some basic HTML examples.
Don't worry if we use tags you have not learned about yet.

HTML documents

All HTML documents must start with a document type declaration: <!DOCTYPE html>

The HTML document itself begins with <html> and ends with </html>

The visible part of the HTML document is between the <body> and the </body>

Let's make a simple HTML document

  
    <!DOCTYPE html>
    <html>
    <head>
    <title>HTML Tutorials</title>
    </head>
    <body>

    <h1>This is a heading</h1>
    <p>This is a paragraph</p>

    </body>
    </html>

Let's explain it

  • The <!DOCTYPE html> declaration defines that the document is HTML5
  • The <html> element is the root element of an HTML page
  • The <head> element contains the meta information about the HTML page
  • The <title> element contains the title of the HTML page, which is shown in the browser title bar
  • The </head> element is the closing tag of the <head> element
  • The <body> element specify the content on the webpage
  • The <h1> element is a heading with the largest size
  • The <p> element is a paragraph
