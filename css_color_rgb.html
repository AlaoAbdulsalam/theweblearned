CSS RGB and RGBA Colors

An RGB Color value represents RED, GREEN and BLUE light sources.

An RGBA color value is an extenstionfor RGB with an Alpha channel (opacity).

RGB Color values

In CSS, a color can be specfied as an RGB value, using this formular:

rgb (red, green, blue)

Each parameter (red, green, and blue) defines the intensity of the color with a value between 0 and 255.

This means that there are 256 x 256 x 256 = 16777216 possible colors

For example, rgb(255, 0, 0) is displayed red because red is set to its highest value (255), and the other two (green and blue) are set to 0.

Another example, rgb(0, 255, 0)is displayed as green, because green is set to its highest value (255), and the other two (red and blue) are set to 0.

To display black, set all color parameters to 0, like this rgb(0, 0, 0).

To display white, set all color parameters to 255, like this: rgb(255, 255, 255).

For example:


rgb(255, 0, 0)


rgb(0, 0, 255)


rgb(50, 80, 10)


rgb(50, 80, 10)


rgb(238, 130, 238)


rgb(90, 0, 255)


rgb(255, 190, 255)


Shades of Gray

Shades of gray often defined using equal values for all three parameters:

For example:


rgb(60, 60, 60)


rgb(90, 90, 90)


rgb(140, 140, 140)


rgb(100, 100, 100)


rgb(160, 160, 160)


rgb(240, 240, 240)


rgb(200, 200, 200)


rgb(200, 200, 200)


RGBA Color values

RGBA color values are an extension of RGB color values with an Alpha channel - which specifies the opacity for a color.

An RGBA color value is specified with:

rgb (red, green, blue, alpha)

The alpha parameter is a number between 0>) (fully transparent) and 1.0 (non transparent at all):

For example:


rgb(255, 99, 71, 0)


rgb(255, 99, 71, 0.2)


rgb(255, 99, 71, 0.6)


rgb(255, 99, 71, 0.8)


rgb(255, 99, 71, 0.4)


rgb(255, 99, 71, 1)
