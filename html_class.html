HTML class attribute

The HTML class attribute is used to specify a class for an HTML element.

Multiple element can share the same class.

How to use the class attribute

The HTML class attribute is usually used to refer to a class name in the style sheet. It can also be used by a JavaScript to access and manipulate elements with the specific class name.

In the example below, we will create a div tag and we will put the class attribute in it putting an attribute value of country

       
         <!DOCTYPE html>
         <html>
         <head>
         <title>HTML Tutorials</title>
          <style>
            .country {
              font-size: 20px;
              color: white;
              background-color: black;
              padding: 15px;
              margin: 10px;
            }
          </style>
         </head>
         <body>

         <div class="country">
           <h2>Uganda</h2>
           <p>Uganda is an african country</p>
         </div>

         <div class="country">
           <h2>Nigeria</h2>
           <p>Nigeria is located in Africa</p>
         </div>

         </body>
         </html>

NOTE: The class element can be used on any HTML element.

NOTE: The class name is case sensitive (Country is different from country)

Multiple classes

HTML elements can belong to more than one class.

To define multiple classes, separate the class names with a space, e.g. <div class="country main">. The element will be styled according to all the classes specified.

In the following example, the <h3> belongs to both the .country class and also to the .main class, and will get the CSS styles from both of the classes: 

       
         <!DOCTYPE html>
         <html>
         <head>
         <title>HTML Tutorials</title>
          <style>
            .country {
              color: black;
              background-color: yellow;
              padding: 15px;
              margin: 10px;
            }

            .main {
              text-align: right;
            }
          </style>
         </head>
         <body>

         <div class="country main">
           <h3>Uganda</h3>
           <p>Uganda is an african country</p>
         </div>

         <div class="country">
           <h3>Nigeria</h3>
           <p>Nigeria is located in Africa</p>
         </div>

         <div class="country">
           <h3>Nigeria</h3>
           <p>Nigeria is located in Africa</p>
         </div>

         </body>
         </html>

Different Elements Can Share Same Class

Example

       
         <!DOCTYPE html>
         <html>
         <head>
         <title>HTML Tutorials</title>
          <style>
            .country {
              font-size: 20px;
              color: white;
              background-color: black;
              padding: 15px;
              margin: 10px;
            }
          </style>
         </head>
         <body>

         <div class="country">
           <h5>Uganda is an african country</h5>
         </div>

         <div class="country">
           <h4>Nigeria is located in Africa</h4>
         </div>

         </body>
         </html>

Chapter summary

  • The HTML class are used in CSS and JavaScript to select and access specific element.
  • The HTML class attribute specifies one or more class names for an element
  • The HTML class attribute can be used on any HTML element
  • The class name is case sensitive
  • JavaScript can access elements with a specific class name with the getElementByClassName() method
