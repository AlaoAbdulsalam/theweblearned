CSS Text Spacing

In this section, you will be learning about all the text spacing in CSS:

Text Spacing - text-indent

The text-indent property is used to add indentation to the first line of a text:

Example:


<!DOCTYPE html>
<html>
<head>
<title>CSS Tutorials</title>
</head>
<style>
  p {
        text-indent: 40px;
    }
</style>
<body>         

<p>Lorem ipsum dolor sit, amet consectetur adipisicing elit. Rem iure quis iusto nam id quaerat, 
quibusdam veritatis. Earum officia possimus, corrupti sequi, architecto debitis libero numquam sint provident ducimus repudiandae. </p>

</body>
</html>

Text Spacing - Letter-spacing

The letter-spacing property is used to add a space between the characters of a text

Example:

    
    <!DOCTYPE html>
    <html>
    <head>
    <title>CSS Tutorials</title>
    </head>
    <style>
        h1 {
            letter-spacing: 5px;
        }
    
    </style>
    <body>         
    
    <h1>This is a heading with a letter spacing of 5 pixels</h1>
    
    </body>
    </html>

Text Spacing - line-height

The line-height property is used to specify space between lines:

Example:

    
    <!DOCTYPE html>
    <html>
    <head>
    <title>CSS Tutorials</title>
    </head>
    <style>
        p {
           line-height: 1.5;
        }
    
    </style>
    <body>         
    
    <p>This is a paragraph with a line height of 1.5 </p>

    </body>
    </html>

Text Spacing - word-spacing

The word-spacing property is used to add a space between the words of a text

Example:

    
    <!DOCTYPE html>
    <html>
    <head>
    <title>CSS Tutorials</title>
    </head>
    <style>
        p {
           word-spacing: 8px;
        }
    
    </style>
    <body>         
    
    <p>This is a paragraph with a word spacing of 8 pixels </p>

    </body>
    </html>
