CSS Text Decoration

In this section, you will be learning about all the text decoration in CSS:

Text Decoration - Line

The text-decoration-line property is used to add a line to a text

You can also add more than one property such as the: line-through and underline

Example:


<!DOCTYPE html>
<html>
<head>
<title>CSS Tutorials</title>
</head>
<style>
    h1 {
        text-decoration: underline;
    }

    h2 {
        text-decoration: overline;
    }

    h3 {
        text-decoration: line-through;
    }

    h4 {
        text-decoration: line-through underline;
    }

</style>
<body>         

<h1>This is a heading </h1>
<h2>This is a heading </h2>
<h3>This is a heading </h3>
<h4>This is a heading </h4>

</body>
</html>

Text Decoration - Color

The text-decoration-color property is used to add a color to a decoration line of a text

Example:

    
    <!DOCTYPE html>
    <html>
    <head>
    <title>CSS Tutorials</title>
    </head>
    <style>
        h1 {
            text-decoration-line: underline;
            text-decoration-color: red;
        }
    
        h2 {
            text-decoration-line: overline;
            text-decoration-color: green;
        }
    
        h3 {
            text-decoration-line: line-through;
            text-decoration-color: yellow;
        }
    
        h4 {
            text-decoration-line: line-through underline;
            text-decoration-color: blue;
        }
    
    </style>
    <body>         
    
    <h1>This is a heading </h1>
    <h2>This is a heading </h2>
    <h3>This is a heading </h3>
    <h4>This is a heading </h4>
    
    </body>
    </html>

Text Decoration - Style

The text-decoration-style property is used to add a style to the decoration line of a text

Example:

    
    <!DOCTYPE html>
    <html>
    <head>
    <title>CSS Tutorials</title>
    </head>
    <style>
        h1 {
            text-decoration-line: underline;
            text-decoration-color: red;
            text-decoration-style: dotted;
        }
    
        h2 {
            text-decoration-line: overline;
            text-decoration-color: green;
            text-decoration-style: solid;
        }
    
        h3 {
            text-decoration-line: line-through;
            text-decoration-color: yellow;
            text-decoration-style: double;
        }
    
        h4 {
            text-decoration-line: line-through underline;
            text-decoration-color: blue;
            text-decoration-style: wavy;
        }

        h5 {
            text-decoration-line: line-through underline;
            text-decoration-color:;
            text-decoration-style: dashed;
        }
    
    </style>
    <body>         
    
    <h1>This is a heading </h1>
    <h2>This is a heading </h2>
    <h3>This is a heading </h3>
    <h4>This is a heading </h4>
    <h5>This is a heading </h5>

    </body>
    </html>

Text Decoration - thickness

The text-decoration-thickness property is used to add a thickness to a decoration line of a text

Example:

    
    <!DOCTYPE html>
    <html>
    <head>
    <title>CSS Tutorials</title>
    </head>
    <style>
        h1 {
            text-decoration-line: underline;
            text-decoration-thickness: auto;
        }
    
        h2 {
            text-decoration-line: overline;
            text-decoration-thickness: 3px;
        }
    
    </style>
    <body>         
    
    <h1>This is a heading </h1>
    <h2>This is a heading </h2>
    
    </body>
    </html>

Text Decoration - None

The text-decoration property also have a value none that remove all text decoration of a text. It is usually used to remove underline on a link

You can also add more than one property such as the: line-through and underline

Example:


<!DOCTYPE html>
<html>
<head>
<title>CSS Tutorials</title>
</head>
<style>
    a {
        text-decoration: none;
    }

</style>
<body>         

<a href="index.html">This is a link </a>

</body>
</html>
