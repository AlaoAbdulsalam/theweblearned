HTML Attribute

  • All HTML elements can have attributes
  • Attributes provide additional information about elements
  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs like: name="value"

There are many type of HTML attribute which we will be explaining one by one:

The href attribute

The href attribute means "hyper-reference. it specifies the URL of a page link.

An example goes thus:

 
   <a href="https://www.salamblog.com">This is a link</a>

Let's explain it:

The <a> is the opening tag, and it is a anchor(link) element.

The https Stands for "Hyper Text Transfer Protocol Secured".

NOTE:
Some website might not have the (s) at the end of their HTTP. This means their website is not secured and it is prone to hackers.

The src attribute

The src attribute is used in an image. Its full meaning is "source". It specifies the path to an image in a website folder.

It does not have an end tag /, just like <br>

An example goes thus:

  
   <img src="images_example.png">

There are 2 ways to specify the URL in the src, which are:

1. Absolute URL:

Links to an external image that is hosted on another website. Example: src="https://www.salam_blog.com/images/img_girl.jpg".

NOTE:
External images might be under copyright. If you do not get permission to use it, you may be in violation of copyright laws. In addition, you cannot control external images; it can suddenly be removed or changed.

2. Relative URL:

Links to an image that is hosted within the website.

NOTE:
Here, the URL does not include the domain name. If the URL begins without a slash, it will be relative to the current page. Example: src="img_girl.jpg". If the URL begins with a slash, it will be relative to the domain. Example: src="/images/img_girl.jpg".

Tips:
It is almost always best to use relative URLs.

The width and height attribute

The <img> tag also uses a width and width attributes. It specifes the width and height of an image.

An example goes thus:

        
           <img src="images_example.png" width="200" height="200">

The alt attribute

The alt attribute is an alternative text for an image that is not showing for some reason. That's why we use the alt attribute

An example goes thus:

            
        <img src="images_example.png" width="200" height="200" alt="A boy operating the computer">

Type of images

  • JPG[JPEG]:

    Joint Photographic Expert Group

    PNG:

    Portable Network Graphics

    GIF:

    Graphic Interchange Format

The style attribute

The style attribute is used to add styles to an element, such as color, font-size, font-family and so on...

An example goes thus:

            
        <h1 style="color:red"></h1>

The lang attribute

The lang attribute is usually included inside the <html> tag to declare the language of the webpage.

The following exapmle specify English as the language:

  
      <!DOCTYPE html>
      <html lang="en">
      <body>

      ...

      </body>
      </html>

Country codes can also be added to the language code in the lang attribute. So, the first two characters define the language of the HTML page, and the last two characters define the country.

The following example specifies English as the language and United States as the country:

  
      <!DOCTYPE html>
      <html lang="en-US">
      <body>

      ...

      </body>
      </html>

The title attribute

The title attribute describe some additional information about an elemnt. The title attribute will be displayed as "tooltip" when you move your mouse over the element on the page.

An example goes thus:

  
    <h1 title="I am a tooltip">This is a tooltip</h1>

Tips:

We Suggest To Always Use Lowercase Attributes. We also sugguest to always surround Attribute values with quotes.

Good:

<img src="images_example.png">

Bad:

<img src=images_example.png>

Chapter summary

  • All HTML elemetns can have "Attributes"
  • The href attribute of <a> specifies the URL of the page the link goes to

  • The src attribute of <img> specifies the path to the image to be displayed

  • The width and height attribute of <img> provide size information for images

  • The alt attribute of <img> provides an alternative text to be displayed for an image

  • The style attribute is used to add styles to an element, such as color, font, size, and more

  • The lang attribute of <html> tag declares the language of the Web page

  • The title attribute defines some extra information about an element
