CSS Background Color

In this section, you will be learning how to add a background color to an webpage or HTML element:

How to add a background color to a webpage

To add a background color to a webpage, you can do it in two ways:

1. Using Inline CSS

To use inline CSS to add background color to a webpage all you need to do is to add the style attribute in the body element(opening tag), after that give it a property name of background-color and a value of the color you want.

Example:

In the example below, we will assign a background color to a page using inline CSS:


<!DOCTYPE html>
<html>
<head>
<title>CSS Tutorials</title>
</head>
<body style="background-color: green;">         

<h1>This is my heading</h1>
<h1>This is another heading</h1>

</body>
</html>

2. Using Internal CSS

To use internal CSS to add background color to a webpage all you need to do is to add the style element in the head section(opening tag), then give it a selector of body and give it a declaration of background-color and yellow.

Example:

In the example below, we will assign a background color to a page using internal CSS:


<!DOCTYPE html>
<html>
<head>
<title>CSS Tutorials</title>
<style>
    body {
        background-color: yellow;
    }
</style>
</head>
<body>         

<h1>This is my heading</h1>
<h1>This is another heading</h1>

</body>
</html>

Opacity

You can also add an opacity to a background color.

Example:


<!DOCTYPE html>
<html>
<head>
<title>CSS Tutorials</title>
<style>
    body {
        background-color: yellow;
        opacity: 0.5;
    }
</style>
</head>
<body>         

<h1>This is my heading</h1>
<h1>This is another heading</h1>

</body>
</html>
