HTML Block and Inline Element

Every HTML element has a default display value, depending on what type of element it is.

There are two display values: Block and Inline

Block-level Elements

A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.

A block-level element always takes up the full width available (streches out to the left and right as far as it can).

Two commonly used block elements are: <p> and <div>

The <p> element defines a paragraph in an HTML document.

The <div> element defines a division or a section in HTML document.

Here are the block-level elements in HTML:

<address> <aside> <canvas> <div> <dt> <figcaption> <footer> <h1>-<h6> <hr> <main> <noscript> <p> <section> <tfoot> <video> <article>
<blockquote> <dd> <dl> <fieldset> <figure> <form> <header> <li> <nav> <ol> <pre> <table> <ul>

Inline Elements

An inline element does not start on anew line.

An inline element only takes up as much width as necessary

This is a <span> element in a paragraph.

Example:

    
      <!DOCTYPE html>
      <html>
      <head>
      <title>HTML Tutorials</title>
    
      </head>
      <body>
     
     <p>This is an inline span <span style="border: 1px solid black;">Hello world</span> element inside a paragraph.</p>

      </body>
      </html>

Here are the Inline elements in HTML:

<address> <a> <acronym> <bdo> <br> <cite> <dfn> <i> <input> <label> <object> <q> <script> <small> <strong> <sup>
<time> <var> <abbr> <b> <big> <button> <code> <em> <img> <kbd> <map> <output> <samp> <select> <span> <sub> <textarea> <tt>

The <div> Element

The <div> element is often used as a container for other HTML elements.

The <div> element has no required attributes, but <style>, <class> and <id> are common.

Example:

    
      <!DOCTYPE html>
      <html>
      <head>
      <title>HTML Tutorials</title>
    
      </head>
      <body>
     
     <div style="background-color: black; color: white; padding:19px;">
      <h2>Nigeria</h2>
      <p>Nigeria is located in West-Africa, Africa. It has more than 2000 ethnic group. Nigeria is divided into four region which are: North, East, West and South.</p>
    </div>

      </body>
      </html>

The <span> Element

The <span> element is an inline element used to mark up a part of a text.

The <span> element has no required attributes, but <style>, <class> and <id> are common.

Example:

    
      <!DOCTYPE html>
      <html>
      <head>
      <title>HTML Tutorials</title>
    
      </head>
      <body>
     
    
      <h2>The span element</h2>
      <p>My mother has a <span style="color: blue;">Blue</span> eye while my father has <span style="color: brown;">brown</span> eye.</p>

      </body>
      </html>

Chapter Summary

  • There are two display value which are: Block and Inline
  • A block-level element always starts on a new line and takes up the full width available
  • An inline element does not start on a new line and it only takes up as much width as necessary
  • The <div> element is a block-level and is often used as a container for other HTML elements
  • The <span> element is an inline container used to mark up a part of a text, or a part of a document
